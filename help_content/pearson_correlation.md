# **Pearson Correlation**

[[Pearson correlation]] measures the strength and direction of the **linear** relationship between two [[continuous variable]]s.

## **How to Use**

1. Go to **Analysis > Correlation**.
2. Select **Pearson** as the method.
3. Choose the columns to include (or leave blank for all numeric columns).

## **Interpreting the Results**

* The [[correlation coefficient]] (**r**) ranges from -1 to +1.
* **r = +1** — perfect positive linear relationship.
* **r = 0** — no linear relationship.
* **r = -1** — perfect negative linear relationship.
* The [[p-value]] tests whether the correlation is significantly different from zero.

## **Assumptions**

* Both variables should be continuous.
* The relationship should be approximately linear (check with a [[scatter plot]]).
* The data should be roughly normally distributed for the p-value to be accurate.

## **Tips**

* If the relationship is monotonic but not linear, consider [[Spearman correlation]] instead.
* [[Outlier]]s can strongly distort Pearson correlations.
