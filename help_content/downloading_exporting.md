# **Downloading & Exporting**

Datagrad allows you to export your data and analysis results in multiple formats.

## **Download Data**

1. Go to **File > Download Data**.
2. Choose **CSV** for a plain-text export or **XLSX** for an Excel workbook.
3. The exported file reflects only the currently active (non-excluded) rows.

## **Download Report**

1. After running an analysis, go to **File > Download Report**.
2. The report is saved as a formatted [[XLSX]] workbook with multiple tabs — one for each result table.

## **Tips**

* Reports include all tables from your most recent analysis run.
* [[CSV]] files do not support multiple sheets; use XLSX if your analysis produced multiple result tables.
